My home projects

First of all, I am happy to share my sewing app, the Sewlist. One of my hobbies is sewing - but since my drawing skills are way behind my creativity in using my fabrics, planning was realy hard. So I made this site, I drew my favourite patterns and built up an environment where I can upload my photoes of my fabrics, or can add them.
It works well on Windows and Android, which were the main purposes.
It uses Firebase as a backend.
You can check the code out at sewlist on GitHub.
